# linux-server-portfolio
# 自宅Linuxサーバー構築の学習記録

## 概要
VirtualBox上にUbuntu 26.04 Server を構築し、各種サービスを設定・運用した学習記録です。

## 構築した環境
- OS: Ubuntu 26.04 LTS
- 仮想化: VirtualBox 7.x

## 習得したスキル

## インフラ構築
- SSH鍵認証・セキュリティー設定
- ファイアウォール(ufw)設定
- Webサーバ(Nginx)構築・HTTPS化
- ファイルサーバ(Samba)構築

### 自動化・運用
- シェルスクリプト作製
- cron による自動バックアップ(rsync)
- Prometheus + Grafanaによる監視ダッシュボード構築

### セキュリティー
- fail2ban による不正アクセス対策
- AppAmor によるプロセスアクセス制限

### コンテナ
- Docker によるコンテナ構築
- Docker Composeによる複数コンテナ管理

### 使用技術
`Linux` `Nginx` `Docker` `Prometheus` `Grafana` `fail2ban` `AppArmor` `Samba` `SSH`
